High School Test Tips

ACT Tips

    General
  • Sleep at least eight hours the night before.
  • Eat good breakfast, like cereal, even if you are not hungry.
      • English
  • Make sure you figure out the independent and dependent clauses in a sentence.
  • Understand the tone and style of the passage in order to select the right keyword.
  • Try to make keep the passage from being repetitive.
  • Make sure that certain phrases have correct spellings.
  • Read the passage at a comfortable speed but fast enough so that it does not lose meaning.
      • Mathematics
  • Make sure you read the problem carefully.
  • Whenever you have to guess and check on a problem, start with the three middle numbers.
  • If you can solve the problem without a calculator, then do the problem without a calculator.
  • If you feel like you are going fast, keep on doing what you are doing to finish the last problems of the test.
      • Reading
  • Make sure you understand the passage.
  • You should get to at least three passages.
  • Accuracy is more important than finishing on time on this test.
      • Science
  • Make sure you distinguish the dependent variable and the independent variable.
  • Make sure you find patterns in each theory when comparing theories.
  • You should try to make it to last section of this test.
